React Practice Projects

This repository contains multiple React projects that I have developed to practice and strengthen my skills in React, hooks, state management, routing, and UI development.

Projects Overview
1. Card Project
A small React project to practice component creation and styling.
Focus on building reusable card components.
2. Employee Management System
A fully functional CRUD app for managing employee data.
Built using React, Bootstrap, and custom CSS.
Features:
Add, update, delete, and view employee details.
Responsive design using Bootstrap.
3. UI of a Webpage
A React project focused on UI development using Tailwind CSS.
Goals:
Build responsive webpage UI components.
Practice modern styling techniques with Tailwind CSS.
4. User App
An application for searching and viewing users.
Features:
React Routing to navigate between pages.
Fetching data from REST APIs.
Search functionality to filter users dynamically.
Clean and responsive UI.
5. React Hooks Practice

This folder contains multiple exercises to practice React Hooks:

a. UseStateHook
A simple counter app using useState.
Features Increase and Decrease buttons to update the counter.
Demonstrates state management in functional components.
b. UseStateInput
Tracks input values using useState.
Practice controlled components and form handling in React.
Technologies Used
React.js
React Hooks (useState, useEffect)
React Router
Tailwind CSS
Bootstrap
REST APIs



Key Learning Outcomes
Component-based architecture
State management with hooks
Routing with React Router
API integration with fetch/axios
Styling with Tailwind CSS and Bootstrap
Controlled components and form handling
